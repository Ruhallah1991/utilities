Flip-Flop Definitions

SR Flip-Flop (Set-Reset Flip-Flop)

This type uses inputs "Set" and "Reset" to either set the output to high, reset it to low, or leave it unchanged.

JK Flip-Flop

An improvement on the SR Flip-Flop, the JK Flip-Flop eliminates the ambiguous state in the SR Flip-Flop by allowing the output to toggle when both J and K inputs are high.

D Flip-Flop (Data Flip-Flop)

This Flip-Flop has a data input (D) and a clock input (CLK). The output Q takes on the value of D when a clock pulse occurs.

T Flip-Flop (Toggle Flip-Flop)

The T Flip-Flop toggles its output when the Toggle input (T) and clock (CLK) are high, making it useful for dividing the input frequency by 2.
